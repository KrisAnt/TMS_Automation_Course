Основные тэги HTML.Парные
Тэг Описание Особенности
<img> Изображение
  • Атрибут src содержит url, откуда берется картинка.
  • Атрибут alt содержит текстб который указывает, что будет выведено при отсутствии картинки
    • <input> Элемент каких-либо действий
  • Атрибут value указыввает начальное значение поля
  • Атрибут type указывает какой тип будет у элемента
    		•
    <br> Переход на новую строку Единичное использование не работает перед элементами, которые автоматически раполагаются на новой строке(например div и p)
    <link> Используется для загрузки в документ других файлов
  • Используется в <head>
  • Атрибут rel отвечает за тип загружаемоего файла
  • Атрибут ref содержит относительный либо абсолютный путь к файлу
    • <a> Ссылка
  • Может содержать закрывающейся тег, текст между тегками будет названием ссылки
  • Атрибут href содержит адрес , куда будет указывать ссылка
    		•
    <table> Таблица
  • Все внутренние теги парные
  • Заголовки таблицы находятся в теге <head>, тело таблицы в <tbody>
  • Каждая строка - внутри <tr>(строка заголовков в том числе) - находятся в thead или tbody
  • В <tr>: ячейка заголовка - <th>. Ячейка таблицы в <td>
    • <ul><li></li></ul> Список
  • Список может быть внутри списка
  • ul-список, каждый пункт списка-li
    		•
    <h1>,<h2>,<h3>,<h4>,<h5>,<h6> Заголовки Заголовок h1 может может быть только 1 на странице
    <title> Название страницы Используется в <head>.Отображает название странице во вкладке браузера
    <select> Комбобокс
  • Каждый элемент комбобокса располагается внутри тэга <select> и имеет тэг </option>
  • Атрибут selected - выбранный элемент по-умолчанию
